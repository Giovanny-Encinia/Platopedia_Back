Si existen problemas con mysql usar los siguientes comandos
mac
export LDFLAGS="-L/usr/local/opt/openssl/lib $LDFLAGS"
export CPPFLAGS="-I/usr/local/opt/openssl/include $CPPFLAGS"

fedora
sudo dnf install python3-devel
sudo dnf install mysql-devel

General(depende de cada OS) 
I had to install these 3 packages first to get it to work for me:

python3-dev
libmysqlclient-dev
libssl-dev

Para más información ver la documentación de mysqlclient
https://pypi.org/project/mysqlclient/
